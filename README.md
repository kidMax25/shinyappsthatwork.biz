# company-website

Built with **ShinyX** - Modern Shiny apps with Shadcn UI

## Getting Started

```bash
# Run the app
Rscript run.R
```

## Project Structure

```
company-website/
├── app/
│   ├── index.html       # Main UI template
│   ├── components/      # Reusable components
│   ├── styles/          # Custom CSS
│   └── js/              # JavaScript files
├── R/
│   ├── server.R         # Shiny server logic
│   └── modules/         # Shiny modules
├── www/                 # Static assets
├── data/                # Data files
└── run.R                # App entry point
```

## Customization

- Edit `app/index.html` for UI changes
- Edit `R/server.R` for server logic
- Add custom styles in `app/styles/custom.css`
- Hot reload is enabled - changes auto-refresh!
